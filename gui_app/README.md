# 职业积分管理系统

一个用Go和Fyne开发的Windows/macOS/Linux桌面应用程序，用于管理游戏职业积分。

## 功能特点

- **欢迎界面**：启动时显示个性化的欢迎对话框
- **独立职业管理**：每个职业（青城QC、仙禽XQ、百花HH、和尚HS、天净TJ、峨眉EM）都有独立的玩家姓名输入框和积分计数器
- **积分控制**：每个职业都有独立的+/-按钮控制积分数值，最小值为0
- **积分复制**：一键复制所有职业的积分信息，格式为：`玩家姓名[职业简称] 积分 | ...`
- **汇总计算**：计算每个职业积分减去其他职业积分总和的盈亏情况
- **金币换算**：积分 × 积分兑换金币比例 = 金币数量
- **界面美化**：职业区块间有分割线，行间距合理，按钮尺寸适中，输入框宽度增加，+/-按钮宽度80像素，兑换比例标签换行显示，所有按钮控件居中对齐，汇总结果区域可滚动，状态提示显示在最下面
- **临时提示**：操作成功后显示3秒自动消失的提示信息
- **结果显示区域**：高度增加300像素，提供更好的结果查看体验
- **输入验证**：自动检测无效的积分输入并提示

## 使用说明

1. **启动程序**：运行程序后会显示欢迎对话框，显示"青衫似故人 最帅，对吗？"
2. **确认进入**：点击"是"按钮进入系统，点击"否"按钮退出程序
3. **输入玩家姓名**：在每个职业区块的输入框中输入玩家姓名
4. **调整积分**：每个职业都有独立的+/-按钮来调整积分数值（最小为0）
5. **复制积分**：点击"复制积分"按钮，格式化后的积分信息将复制到剪贴板，界面显示3秒成功提示
6. **设置兑换比例**：在"积分兑换金币比例"输入框中设置兑换比例（默认1.0）
7. **查看汇总**：点击"汇总计算"按钮，计算每个职业积分减去其他职业积分总和的盈亏情况，界面显示3秒成功提示

## 复制格式

### 积分复制格式：
```
玩家A[青城QC] 100 | 玩家B[仙禽XQ] 200 | 玩家C[百花HH] 150
```

### 汇总结果格式：
```
青城QC: 积分 100.00 - 其他职业总和 450.00 = 积分盈亏 -350.00, 金币盈亏 -350.00
仙禽XQ: 积分 200.00 - 其他职业总和 400.00 = 积分盈亏 -200.00, 金币盈亏 -200.00
百花HH: 积分 150.00 - 其他职业总和 400.00 = 积分盈亏 -250.00, 金币盈亏 -250.00
和尚HS: 职业与玩家未绑定
天净TJ: 积分 80.00 - 其他职业总和 470.00 = 积分盈亏 -390.00, 金币盈亏 -390.00
峨眉EM: 积分 600.00 - 其他职业总和 -50.00 = 积分盈亏 +650.00, 金币盈亏 +650.00
```

*说明：计算方式为本职业积分 × (职业总数-1) - 其他职业积分总和，正数表示盈利，负数表示亏损，然后乘以金币兑换比例。输入框为玩家姓名，计数器为积分数值。*

## 技术实现

- **GUI框架**：Fyne v2 - 跨平台Go GUI库
- **语言**：Go 1.20+
- **平台支持**：Windows, macOS, Linux

## 运行要求

- Go 1.20 或更高版本
- 已安装Fyne依赖

## 安装依赖

```bash
go mod tidy
```

## 运行程序

```bash
go run main.go
```

## 构建可执行文件

```bash
go build -o 职业积分管理系统 main.go
```

## 注意事项

- 确保输入的积分和兑换比例为有效的数字
- 空输入框将被忽略在计算中
- 所有结果会自动复制到系统剪贴板
